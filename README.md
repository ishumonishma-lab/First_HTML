Event Management Website

This project is a simple Event Management Website developed using HTML and CSS in Visual Studio Code. The website displays information about upcoming events, a gallery section, and important event details in a clean and organized layout.

 Project Description

The website is designed to showcase upcoming events and provide visitors with useful information about event activities. It includes sections such as event announcements, photo gallery, and event details. The design focuses on simplicity and readability so users can easily explore the event content.

Features

* Upcoming Events section
* Photo Gallery display
* Event information and description
* Clean and simple user interface
* Structured layout using HTML and CSS

 Technologies Used

* HTML5
* CSS3
* Visual Studio Code

 Project Sections

Upcoming Events: Displays announcements and updates about the event.

Photo Gallery: Shows images related to the event and creative visuals.

Event Details: Includes information such as free entry, food availability, and rules for visitors.
